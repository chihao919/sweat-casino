## Season 1 比賽規則

### 基本規則
- **賽期**：6/30（一）~ 7/27（日），共 4 週
- **隊伍**：紅牛隊 🐂 vs 白熊隊 🐻‍❄️
- **勝負判定**：4 週累計總距離 × 活動參與率，高者獲勝

### 每週節奏

| 時間 | 事件 |
|------|------|
| 每天 | 跑步 → 自動同步 → 賺 SC |
| 每天 | 在 LINE 群曬成績、互相激勵 |
| 每週一 00:00 | 生存稅結算（跑不到 20km 扣 10%） |
| 每週日 23:59 | 週統計快照、隊伍戰況更新 |

### 每日成績公布
- 鼓勵參賽者每天在 LINE 群分享今天跑了多少
- 每天在群內公布當日排行 Top 5
- 每週一公布週排行 + 隊伍戰況

### 獎勵結構

**個人獎項：**
- 🥇 總距離第一名
- 🥇 總 SC 第一名
- 🥇 最長連續天數
- 🏆 最佳進步獎（跟暖身週相比進步最多）

**隊伍獎項：**
- 獲勝隊伍全員獲得 Season Bonus SC
